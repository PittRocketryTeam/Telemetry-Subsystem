# test-data

## Format
Each row in the file `data.csv` is a packet.<br>
Health packets have an id of 1, Orientation packets have an id of 2.

## Health Packet
Data points are in the following order:
- id
- time
- vbat
- vr5
- vr3
- tbat
- current

## Orientation Packet
Data points are in the following order
- id
- time
- lon
- lat
- ax
- ay
- az
- vy
- altitude
- omega_x
- omega_y
- omega_z
- quat_w
- quat_x
- quat_y
- quat_z
